# Bodh

A Next.js application with voice chatbot capabilities using LiveKit Agents.

## Getting Started

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## MCP Server Setup

This project uses MCP (Model Context Protocol) servers for enhanced development experience:

### Figma MCP Server
For working with Figma designs and components:
- [Figma MCP Server Remote Installation Guide](https://developers.figma.com/docs/figma-mcp-server/remote-server-installation/)
